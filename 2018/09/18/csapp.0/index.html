计算机系统的硬件组成

对基本的硬件组成要有理解，计算机的硬件组成主要有 CPU,内存，磁盘，图像适配器，网络设配器以及某些USB控制器等

cpu

中央处理单元是解释存储在主存中指令的引擎，在任何时刻，程序计数器都指向一条指令，处理器一直取指令，解析指令的位，执行指令的操作，更新下一条指令，可以看做是这样一个简单的模型。

CPU在指令的要求下一般会做这样几件事:

  1. 加载: 从主存复制两个字节到寄存器，覆盖原来寄存器的值
  2. 把某一个字存储到内存的某个地址
  3. 把两个寄存器里面的值复制到ALU，做加法或者乘法操作，把结果放到一个寄存器
  4. 从指令中抽取一个字，更新程序计数器(PC)
cpu的行为

抽象

计算机的世界有这么几个很关键的抽象:

  • 指令集提供了对 CPU硬件的抽象，使用这个抽象，机器代码就好像运行在一次只执行一条指令的处理器上，实际的硬件比描述的要复杂的多，它并行的执行多条指令，但又和这个简单的模型保持一致
  • 文件是对 IO的抽象
  • 虚拟存储器是对程序存储器的抽象
  • 进程是对一个 正在运行的程序的抽象
